  <action>
Create a new `DictusKeyboard/TextPrediction/` directory with 3 files:

**FrequencyDictionary.swift:**
- Struct with private `ranks: [String: Int]` dictionary (lower rank = more common)
- `mutating func load(language: String)` -- loads `{language}_frequency.json` from Bundle.main. Use `JSONDecoder` to decode `[String: Int]`. Handle missing file gracefully (empty dict).
- `func rank(of word: String) -> Int` -- returns rank for lowercased word, or `Int.max` if not found
- Only load the active language, not both. Lazy: load on first call or language change.

**TextPredictionEngine.swift:**
- Class (not struct -- holds UITextChecker instance which is a reference type)
- Properties: `private let textChecker = UITextChecker()`, `private var frequencyDict = FrequencyDictionary()`, `private var language: String = "fr"`
- `func setLanguage(_ lang: String)` -- updates language, reloads frequency dict. For UITextChecker language param, use "fr" for French, "en" for English (verify against UITextChecker.availableLanguages at init and log if not found).
- `func suggestions(for partialWord: String) -> [String]` -- returns up to 3 ranked word completions:
  1. Call `textChecker.completions(forPartialWordRange:in:language:)` with the full partial word as the string and range covering all of it
  2. Sort results by `frequencyDict.rank(of:)` ascending (lower rank = more common = first)
  3. Return `Array(ranked.prefix(3))`
  4. If completions is empty or nil, return empty array
- `func spellCheck(_ word: String) -> String?` -- returns the best correction for a misspelled word, or nil if correctly spelled:
  1. Call `textChecker.rangeOfMisspelledWord(in:range:startingAt:wrap:language:)`
  2. If `misspelled.location == NSNotFound`, return nil (word is correct)
  3. Call `textChecker.guesses(forWordRange:in:language:)`
  4. Return the first guess ranked by frequency dict (not just `.first` -- re-rank guesses too)
- `func accentSuggestions(for partialWord: String) -> [String]?` -- returns accent variants for single-char vowel input:
  1. Guard `partialWord.count == 1` else return nil
  2. Look up `AccentedCharacters.mappings[partialWord.lowercased()]`
  3. If found, build array: `[partialWord] + variants.prefix(2)` (max 3 slots)
  4. Preserve case: if input is uppercase, uppercase the variants too
  5. Return the 3-element array, or nil if not a vowel

**SuggestionState.swift:**
- `enum SuggestionMode { case idle, completions, accents }`
- `class SuggestionState: ObservableObject` with:
  - `@Published var suggestions: [String] = []` (the 3 slots, can be fewer)
  - `@Published var mode: SuggestionMode = .idle`
  - `@Published var currentWord: String = ""` (the word being completed, needed for replacement)
  - `private let engine = TextPredictionEngine()`
  - `var autocorrectEnabled: Bool` -- reads from AppGroup.defaults SharedKeys.autocorrectEnabled
  - `var lastAutocorrect: AutocorrectState?` -- tracks the last autocorrection for undo
  - `func update(proxy: UITextDocumentProxy)` -- called on each keystroke:
    1. Extract current partial word from `proxy.documentContextBeforeInput` using `String.enumerateSubstrings(options: .byWords)` to find the last word. Only if text does NOT end with space/newline.
    2. If partial word is a single accentable vowel (first char after space), switch to accent mode via `engine.accentSuggestions(for:)`
    3. Otherwise, get completions via `engine.suggestions(for:)`
    4. Update `suggestions`, `mode`, `currentWord`
    5. If no partial word (after space, empty field), set mode to `.idle`, clear suggestions
  - `func clear()` -- resets to idle state
  - `func setLanguage(_ lang: String)` -- forwards to engine
- `struct AutocorrectState` with: `originalWord: String`, `correctedWord: String`, `insertedSpace: Bool`

**SharedKeys.swift update:**
- Add `public static let autocorrectEnabled = "dictus.autocorrectEnabled"` to SharedKeys enum

  <action>
Create frequency dictionary JSON files for French and English. Format: `{"word": rank}` where lower rank = more common.

**French (fr_frequency.json):**
Generate a JSON object with the top 15,000-20,000 most common French words ranked by frequency. Use known French word frequency data (Lexique 3.83 / standard French frequency lists). The file should include:
- Common articles: de(1), la(2), le(3), les(4), un(5), une(6), des(7)...
- Common verbs: est(8), a(9), sont(10), fait(11), peut(12), avoir(13), etre(14)...
- Common nouns, adjectives, adverbs, prepositions, conjunctions
- Accented forms: cafe should NOT be in the dict, but cafe-with-accent (cafe) SHOULD be (this helps autocorrect fix missing accents)
- Include both infinitive and common conjugated forms
- Target file size: under 1.5MB

**English (en_frequency.json):**
Same format, top 15,000-20,000 most common English words. Include:
- Common words: the(1), be(2), to(3), of(4), and(5), a(6)...
- Target file size: under 1.5MB

**Practical approach:** Since generating 20K word frequency data from scratch is impractical in a code editor, create a script that:
1. Uses a curated list of the top ~5,000 most frequent French/English words (hardcoded in the script from well-known frequency lists)
2. Generates the JSON files
3. For French, ensure all common accented words are included (important for autocorrect)

If 5K words produces a file well under 1.5MB, that's fine -- UITextChecker provides the completions, the frequency dict only ranks them. Even 2,000-3,000 common words would cover the vast majority of ranking needs.

**Bundle in Xcode:** Add both JSON files to the DictusKeyboard target's "Copy Bundle Resources" build phase in project.pbxproj. Search for existing resource entries (like Info.plist) to follow the pattern. The files must be in the DictusKeyboard bundle for `Bundle.main.url(forResource:withExtension:)` to find them at runtime.

After creating the files, delete any generation script used (clean workspace).
