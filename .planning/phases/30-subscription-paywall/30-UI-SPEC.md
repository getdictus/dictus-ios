# Phase 30 — UI Design Contract

> Visual and interaction contract for the subscription paywall, Pro banner, and Settings Pro section. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (native SwiftUI) |
| Preset | not applicable |
| Component library | SwiftUI native + DictusCore Design system |
| Icon library | SF Symbols (system) |
| Font | SF Pro Rounded (headings) + SF Pro Text (body) — system fonts |

Source: `DictusCore/Sources/DictusCore/Design/DictusTypography.swift`, `DictusColors.swift`, `GlassModifier.swift`

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4pt | Icon-label gaps, inline padding |
| sm | 8pt | Inner card spacing (label to text), compact element spacing |
| md | 16pt | Default element padding, card internal padding, corner radius |
| lg | 24pt | VStack section spacing (established pattern in HomeView) |
| xl | 32pt | Top/bottom page padding |
| 2xl | 48pt | Hero section vertical breathing room |

Exceptions: 12pt used for bottom links vertical spacing (restore + ToS/Privacy cluster)

## Typography

| Role | Font | Size | Weight | Line Height | SwiftUI Token |
|------|------|------|--------|-------------|---------------|
| Display | SF Pro Rounded | .largeTitle | .bold | 1.2 | `.system(.largeTitle, design: .rounded, weight: .bold)` |
| Heading | SF Pro Rounded | .title | .bold | 1.2 | `.dictusHeading` |
| Subheading | SF Pro Rounded | .title3 | .semibold | 1.3 | `.dictusSubheading` |
| Body | SF Pro Text | .body | .regular | 1.5 | `.dictusBody` |
| Caption | SF Pro Text | .caption | .regular | 1.4 | `.dictusCaption` |

Usage in this phase:
- **Display**: Paywall hero title "Dictus Pro" (only instance of .largeTitle in phase)
- **Heading**: Not used in paywall (reserved for other screens)
- **Subheading**: Feature card titles, CTA button label, Pro banner title
- **Body**: Feature card descriptions, beta banner text
- **Caption**: "Cancel anytime" text, restore link, ToS/Privacy links, Settings Pro row subtitle, BETA pill label

### 5. Settings Pro Section

#### "Dictus Pro" row (top of Settings List)

| Element | Spec |
|---------|------|

| Icon | `crown.fill` tinted `Color.dictusAccent` |
| Label | "Dictus Pro" in default List row style |

| Badge (pro active) | `checkmark.circle.fill` in `Color.dictusSuccess` |

#### "Pro Features" section (new section in Settings)

| Element | Spec |
|---------|------|
| Section header | "Pro Features" |
| Row pattern | Each Pro feature as a row with: lock icon + "PRO" pill when locked, toggle when unlocked |

| Unlocked row | Toggle with @AppStorage binding to per-feature SharedKey |
| Features listed | Smart Mode, History, Vocabulary (matching ProFeature enum order) |

### 6. Keyboard Extension (PAY-05 redefined)

| Element | Spec |

## Interaction States

### Subscribe CTA Button

| State | Visual |
|-------|--------|
| Default | `Color.dictusAccent` background, white text, 1.0 opacity |
| Pressed | Scale 0.92, opacity 0.85 (via `GlassPressStyle`) |
| Loading (purchasing) | ProgressView() spinner replaces button text, button disabled |
| Success | Button disappears, paywall auto-dismisses after 0.5s delay |
| Failed | Button returns to default, error alert shown |
| Disabled (no products) | 0.5 opacity, non-interactive |

### Pro Banner (HomeView)

| State | Visual |
|-------|--------|
| Default | Visible with gradient background |
| Pressed | Scale 0.97 (via `GlassPressStyle`) |
| Pro active | Not rendered (conditional `if !proStatus.isProActive`) |

### Feature Cards (Paywall)

| State | Visual |
|-------|--------|
| Default | Static display, non-interactive |
| No interaction states | Cards are informational only, not tappable |

### Settings Pro Row

| State | Visual |
|-------|--------|
| Beta active | Crown icon + "BETA" pill badge |
| Free (not beta) | Crown icon + disclosure chevron |
| Pro active | Crown icon + green checkmark |

## Transitions & Animation

| Transition | Spec |
|------------|------|
| Paywall push | Default NavigationStack push animation (right-to-left slide) |
| Paywall dismiss after purchase | `dismiss()` environment action after 0.5s delay |
| Purchase spinner | `ProgressView()` with default SwiftUI animation |
| Pro banner disappearance | `.transition(.opacity.combined(with: .scale))` with `.animation(.easeOut(duration: 0.3))` |
| Settings badge change | Default SwiftUI implicit animation |

## Accessibility

| Element | Requirement |
|---------|-------------|
| Feature cards | Combined accessibility label: "{title}: {description}" |
| Subscribe CTA | accessibilityLabel: "Subscribe to Dictus Pro for {displayPrice} per month" |
| Restore link | accessibilityLabel: "Restore previous purchases" |
| PRO pill badge | accessibilityLabel: "Pro feature" (on Settings rows) |
| BETA pill badge | accessibilityLabel: "Beta — all features free" |
| Lock icon | Decorative — accessibilityHidden(true), semantics carried by PRO pill |
| Dynamic Type | All text uses system text styles (.title, .title3, .body, .caption) — automatically scales |
| VoiceOver order | Hero title > Feature cards (top to bottom) > CTA/Beta banner > Bottom links |
