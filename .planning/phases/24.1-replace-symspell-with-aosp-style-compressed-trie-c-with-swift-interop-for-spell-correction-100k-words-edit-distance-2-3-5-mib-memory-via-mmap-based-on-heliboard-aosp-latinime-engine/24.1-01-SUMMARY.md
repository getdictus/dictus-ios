## Decisions Made
- Used existing 40K words per language rather than attempting external data download for 100K expansion (documented expansion steps in plan notes)
- Log-scale normalization preserves frequency ranking while fitting uint16 range
- Capped header max_freq to uint32 for English (Google N-gram counts are int64)
- Scorer uses static recursive function to avoid virtual dispatch overhead in hot path
