## Decisions Made
- Used auto-generated keys from English source text (Xcode default) instead of manual key naming -- simpler, zero maintenance overhead
- Localized model descriptions at the view layer (ModelInfo+Localized.swift) instead of adding localization infrastructure to DictusCore Swift Package -- keeps DictusCore as pure data/logic layer
- Included accessibility label localization in this pass (1 string: "Stop recording")
- Language names in the transcription picker ("Français", "English") are NOT localized -- they are language names that should always display in their native form
