**Step 3: Create ModelInfo+Localized.swift**
Create `DictusApp/Extensions/ModelInfo+Localized.swift` (create Extensions/ directory if needed):

```swift
// DictusApp/Extensions/ModelInfo+Localized.swift
// Localized display strings for ModelInfo, resolved via String Catalog.
import DictusCore
import SwiftUI

extension ModelInfo {
    /// Localized description for display in model cards.
    /// Source strings are in English; French provided via String Catalog.
    var localizedDescription: String {
        switch identifier {
        case "openai_whisper-tiny", "openai_whisper-base":
            return String(localized: "Fast but inaccurate")
        case "openai_whisper-small":
            return String(localized: "Accurate and balanced")
        case "openai_whisper-small_216MB":
            return String(localized: "Compact and fast")
        case "openai_whisper-medium":
            return String(localized: "Best accuracy")
        case "parakeet-tdt-0.6b-v3":
            return String(localized: "Fast and accurate (NVIDIA)")
        default:
            return description
        }
    }
}
```

**Step 5: Create DictusApp/Localizable.xcstrings**

Create the String Catalog JSON file with `sourceLanguage: "en"` and French translations for every English string. The .xcstrings format is a JSON file. Include every string from the replacement table above with its French translation.

The format is:
```json
{
  "sourceLanguage": "en",
  "strings": {
    "English key": {
      "localizations": {
        "fr": {
          "stringUnit": {
            "state": "translated",
            "value": "French translation"
          }
        }
      }
    }
  },
  "version": "1.0"
}
```

For strings with interpolation (like `"Delete %@? The model will be removed from your device."` and `"%lld%%"`), use the appropriate format specifiers.
