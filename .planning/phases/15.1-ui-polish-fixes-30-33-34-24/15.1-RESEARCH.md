**AudioServicesPlaySystemSound with custom WAV files:**
```swift
import AudioToolbox

// Register a custom sound from bundle
var soundID: SystemSoundID = 0
if let url = Bundle.main.url(forResource: "electronic_01a", withExtension: "wav") {
    AudioServicesCreateSystemSoundID(url as CFURL, &soundID)
    AudioServicesPlaySystemSound(soundID)
}
```

Key facts:
- `AudioServicesCreateSystemSoundID` registers a WAV file and returns a SystemSoundID
- `AudioServicesPlaySystemSound` plays it (respects silent switch natively)

- Sound files are at `/Users/pierreviviere/dev/dictus/Dictus/Sounds/` (29 WAV files)
- No AVAudioSession conflict -- AudioServicesPlaySystemSound bypasses AVAudioSession entirely

### Anti-Patterns to Avoid
- **Using AVAudioPlayer for sound feedback:** Will conflict with WhisperKit's AVAudioSession. AudioServicesPlaySystemSound operates independently.
- **Caching SystemSoundIDs in an init:** The 29 sounds do not all need to be registered upfront. Register on first play, cache the ID, dispose on deinit.
- **Hardcoding pixel positions for button alignment:** Use the same padding/frame values as ToolbarView -- if ToolbarView changes later, both stay in sync.

## Common Pitfalls

### Pitfall 1: AudioServicesCreateSystemSoundID memory leak
**What goes wrong:** Creating SystemSoundID without disposing leads to resource leak
**Why it happens:** Each AudioServicesCreateSystemSoundID allocates system resources
**How to avoid:** Cache SystemSoundIDs in SoundFeedbackService, dispose all in deinit with AudioServicesDisposeSystemSoundID
**Warning signs:** Increasing memory usage after repeated sound playback

### Pitfall 2: Sound timing with WhisperKit audio session
**What goes wrong:** Start sound gets cut off when WhisperKit calls setCategory/setActive
**Why it happens:** WhisperKit's startRecordingLive() overrides audio session internally
**How to avoid:** Play start sound BEFORE configureAudioSession(), add 200-300ms DispatchQueue.main.asyncAfter delay before starting recording. AudioServicesPlaySystemSound is fire-and-forget so it completes even if session changes.
**Warning signs:** Start sound is silent or truncated

### Pitfall 3: RecordingOverlay height mismatch after button resize
**What goes wrong:** Waveform jumps vertically when transitioning to/from transcribing state
**Why it happens:** transcribingContent reserves `Color.clear.frame(height: 44)` for the top bar area -- must match new button height
**How to avoid:** Update ALL three content states (requested, recording, transcribing) to use consistent heights after button resize
**Warning signs:** Visual Y-axis jump during recording -> transcribing transition

### Pitfall 4: CFBundleDisplayName vs CFBundleName
**What goes wrong:** App still shows "DictusApp" after adding CFBundleDisplayName
**Why it happens:** Build cache or Simulator needs clean build
**How to avoid:** Clean build folder (Cmd+Shift+K) after Info.plist changes, delete app from Simulator and reinstall
**Warning signs:** Old name persists after change

### Pitfall 5: Dynamic Island proportions wrong on mockup
**What goes wrong:** Dynamic Island looks too big or too small
**Why it happens:** Real device proportions don't scale linearly to 120pt mockup
**How to avoid:** Use proportional scaling: real DI is ~126x37 points on 393pt-wide screen = 32% width. On 120pt mockup = ~38pt wide, ~11pt tall
**Warning signs:** DI dominates the mockup or is barely visible

## Code Examples

### SoundFeedbackService Pattern
```swift
// Source: follows HapticFeedback.swift pattern + AudioToolbox API
import AudioToolbox

enum SoundFeedbackService {
    private static var cachedSounds: [String: SystemSoundID] = [:]

    private static func isEnabled() -> Bool {
        let defaults = UserDefaults(suiteName: AppGroup.identifier)
        return defaults?.object(forKey: SharedKeys.soundFeedbackEnabled) as? Bool ?? true
    }

    static func play(_ soundName: String) {
        guard isEnabled(), !soundName.isEmpty else { return }

        if let cached = cachedSounds[soundName] {
            AudioServicesPlaySystemSound(cached)
            return
        }

        guard let url = Bundle.main.url(forResource: soundName, withExtension: "wav") else { return }
        var soundID: SystemSoundID = 0
        let status = AudioServicesCreateSystemSoundID(url as CFURL, &soundID)
        guard status == kAudioServicesNoError else { return }

        cachedSounds[soundName] = soundID
        AudioServicesPlaySystemSound(soundID)
    }

    static func playRecordStart() {
        let defaults = UserDefaults(suiteName: AppGroup.identifier)
        let name = defaults?.string(forKey: SharedKeys.recordStartSoundName) ?? "electronic_01a"
        play(name)
    }

    static func playRecordStop() {
        let defaults = UserDefaults(suiteName: AppGroup.identifier)
        let name = defaults?.string(forKey: SharedKeys.recordStopSoundName) ?? "ui_chime_01"
        play(name)
    }

    static func playRecordCancel() {
        let defaults = UserDefaults(suiteName: AppGroup.identifier)
        let name = defaults?.string(forKey: SharedKeys.recordCancelSoundName) ?? "electronic_02a"
        play(name)
    }
}
```
