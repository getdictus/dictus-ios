# Milestone v1.3: Public Beta

**Status:** SHIPPED 2026-04-07
**Phases:** 17-22

## Overview

Complete UIKit keyboard rebuild using giellakbd-ios, advanced touch interactions, feature reintegration, memory optimization, and public TestFlight launch. First public beta distributed.

## Milestone Summary

**Key Decisions:**
- Vendor giellakbd-ios source directly (not SPM) for full control over keyboard behavior
- Hybrid UIKit keyboard + SwiftUI toolbar architecture (best of both worlds)
- DragGesture replaced by UICollectionView direct touch handling (eliminates dead zones)
- Category pagination for emoji picker (memory-safe, 50 MiB budget)
- Per-target privacy manifests (minimum API declarations per target)
- Tester limit set to 250 for initial public beta
- GitHub Release tagged v1.3.0-beta as pre-release for social media announcement

**Issues Resolved:**
- Dynamic Island stuck on REC state (watchdog + state machine)
- Export logs slow on large files (O(1) size-based trim)
- Keyboard dead zones (UICollectionView architecture)
- Edge key touch delay (UIWindow gesture recognizer fix)
- Emoji picker 139 MiB memory bloat (category pagination)

**Issues Deferred:**
- COLD-03: Cold start auto-return to source app (no public iOS API exists)

**Technical Debt Incurred:**
- None significant
