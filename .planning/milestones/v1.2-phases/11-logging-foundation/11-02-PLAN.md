  <name>Task 1: Instrument all 6 subsystems with structured log events</name>
  <files>
    DictusApp/DictusApp.swift,
    DictusApp/DictationCoordinator.swift,
    DictusApp/Audio/AudioRecorder.swift,
    DictusApp/Audio/TranscriptionService.swift,
    DictusApp/Models/ModelManager.swift,
    DictusKeyboard/KeyboardViewController.swift
  </files>
  <action>
    Add PersistentLog.log(.event) calls at key points in each subsystem. Target ~5-10 entries per dictation session (key events only, no per-frame/per-buffer logging per user decision).

    **DictusApp.swift (lifecycle subsystem):**
    - `init` or `body` scene setup: `PersistentLog.log(.appLaunched(version: Bundle.main... ))`.
    - Use `scenePhase` changes (or UIApplicationDelegate if already in use) to log: `.appDidBecomeActive`, `.appWillResignActive`, `.appDidEnterBackground`.
    - After WhisperKit loads: `.appWhisperKitLoaded(modelName:)`.

    **DictationCoordinator.swift (dictation subsystem):**
    - Migrate ALL existing `PersistentLog.log("...")` calls (~10) to structured events.
    - At dictation start: `.dictationStarted(fromURL:appState:engineRunning:)`.
    - At dictation complete: `.dictationCompleted(durationMs:)`.
    - At dictation fail: `.dictationFailed(error:)` -- use error.localizedDescription (WhisperKit errors are safe per user decision).
    - At deferred dictation: `.dictationDeferred(reason:)`.
    - Remove all old `PersistentLog.log("free text")` calls -- replace each with the appropriate structured event.

    **AudioRecorder.swift (audio subsystem):**
    - Engine start: `.audioEngineStarted`.
    - Engine stop: `.audioEngineStopped`.
    - Session configured: `.audioSessionConfigured(category:)`.
    - Session failure: `.audioSessionFailed(error:)`.

    **TranscriptionService.swift (transcription subsystem):**
    - Start: `.transcriptionStarted(modelName:)`.
    - Complete: `.transcriptionCompleted(durationMs:wordCount:)` -- compute duration from start time, count words in result (NOT logging the text itself).
    - Fail: `.transcriptionFailed(error:)`.

    **ModelManager.swift (model subsystem):**
    - Download start: `.modelDownloadStarted(name:sizeMB:)`.
    - Download complete: `.modelDownloadCompleted(name:)`.
    - Download fail: `.modelDownloadFailed(name:error:)`.
    - Model selected: `.modelSelected(name:)`.
    - Compilation start/complete if applicable.

    **KeyboardViewController.swift (keyboard subsystem):**
    - `viewDidAppear`: `.keyboardDidAppear`.
    - `viewDidDisappear`: `.keyboardDidDisappear`.
    - Mic button tap handler: `.keyboardMicTapped`.
    - After text insertion: `.keyboardTextInserted` (no content parameter!).

    Import DictusCore where not already imported. Keep logging lightweight -- one line per event, no conditional logging (all levels always active per user decision).
