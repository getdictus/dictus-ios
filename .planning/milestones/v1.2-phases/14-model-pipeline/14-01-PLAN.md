2. Add two new public static methods after `forIdentifier()`:

```swift
/// Returns the recommended model identifier based on device RAM.
/// >=6 GB RAM (iPhone 12 Pro, 13 Pro, 14+, 15+, 16+) -> Parakeet v3
/// <=4 GB RAM (iPhone 12, 12 mini, 13, 13 mini) -> Whisper Small
public static func recommendedIdentifier() -> String {
    let ramGB = ProcessInfo.processInfo.physicalMemory / 1_073_741_824
    if ramGB >= 6 {
        return "parakeet-tdt-0.6b-v3"
    } else {
        return "openai_whisper-small"
    }
}

/// Whether the given model identifier matches the device-recommended model.
public static func isRecommended(_ identifier: String) -> Bool {
    identifier == recommendedIdentifier()
}
```

3. Add a log statement for RAM detection so it can be diagnosed on device:
```swift
// Inside recommendedIdentifier(), after computing ramGB:
PersistentLog.log(.info, subsystem: .modelManager, message: "Device RAM: \(ramGB) GB, recommending: \(ramGB >= 6 ? "parakeet" : "small")")
```
Note: Check if PersistentLog is importable in DictusCore. If not (it likely is since DictusCore defines it), skip the log and add it in ModelManager instead.

Why `ModelInfo` not `ModelManager`: This is catalog-level logic (which model fits this device). It doesn't depend on download state or any @Published properties. Accessible from both ModelManager and ModelDownloadPage without passing around an ObservableObject.
