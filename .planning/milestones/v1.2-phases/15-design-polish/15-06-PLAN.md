In ModelManagerView.swift, update the `downloadedModels` computed property (L40-41) to also include models that are currently downloading or prewarming:

```swift
private var downloadedModels: [ModelInfo] {
    ModelInfo.allIncludingDeprecated.filter { model in
        let state = modelManager.modelStates[model.identifier] ?? .notDownloaded
        switch state {
        case .downloading, .prewarming, .ready, .error:
            return true
        case .notDownloaded:
            return modelManager.downloadedModels.contains(model.identifier)
        }
    }
}
```

Also update `availableModels` to exclude models that are downloading/prewarming:

```swift
private var availableModels: [ModelInfo] {
    ModelInfo.all.filter { model in
        let state = modelManager.modelStates[model.identifier] ?? .notDownloaded
        switch state {
        case .downloading, .prewarming, .ready, .error:
            return false
        case .notDownloaded:
            return !modelManager.downloadedModels.contains(model.identifier)
        }
    }
}
```

This ensures that when a user taps a model in Available, it immediately moves to the Downloaded section showing its download progress.
