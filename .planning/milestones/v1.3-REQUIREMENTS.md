# Requirements: Dictus v1.3 Public Beta (ARCHIVED)

**Defined:** 2026-03-27
**Archived:** 2026-04-07
**Core Value:** A user can dictate text in French in any iOS app and correct it immediately on the same keyboard -- no subscription, no cloud, no account.

## v1.3 Requirements

### Bug Fixes (Phase 17)

- [x] **FIX-01**: Dynamic Island never gets stuck on REC state after recording ends or is cancelled
- [x] **FIX-02**: Export logs completes within a few seconds and shows a spinner during export

### Keyboard Architecture (Phase 18)

- [x] **KBD-01**: User can type any character on the AZERTY keyboard with no dead zones
- [x] **KBD-02**: User can switch to QWERTY layout with zero-dead-zone behavior
- [x] **KBD-03**: User can toggle shift (single tap) and caps lock (double tap) with visual state changes
- [x] **KBD-04**: User can switch between letters, numbers, and symbols layers
- [x] **KBD-05**: User can hold backspace and characters delete with accelerating repeat speed
- [x] **KBD-06**: User gets autocapitalization after sentence-ending punctuation
- [x] **KBD-07**: Double-space inserts period followed by space
- [x] **KBD-08**: User sees key popup preview on press

### Feel & Polish (Phases 18-19)

- [x] **FEEL-01**: User feels haptic feedback on touchDown
- [x] **FEEL-02**: User hears 3-category key sounds respecting silent switch
- [x] **FEEL-03**: Key popup preview on press (provided by giellakbd-ios)
- [x] **FEEL-04**: User can long-press vowels to see French accent characters and drag to select
- [x] **FEEL-05**: User can drag spacebar to move cursor with haptic ticks
- [x] **FEEL-06**: Edge keys produce haptic feedback and popup on touchDown, not touchUp

### Dictation Reintegration (Phase 20)

- [x] **DICT-01**: User can tap mic button to start recording and sees recording overlay
- [x] **DICT-02**: User gets transcription auto-inserted at cursor after recording
- [x] **DICT-03**: User sees Full Access banner when permissions needed
- [x] **DICT-04**: User sees 3-slot suggestion bar with French autocorrect

### Prediction & Settings (Phase 20)

- [x] **PRED-01**: Suggestion bar shows French autocorrect suggestions
- [x] **PRED-02**: User can tap suggestion to insert
- [x] **PRED-03**: User can undo autocorrect with backspace
- [x] **SET-01**: User can select default opening layer (letters or numbers) with live preview

### TestFlight (Phase 22)

- [x] **TF-01**: App passes Beta App Review with Privacy Manifests and Full Access justification
- [x] **TF-02**: External testing group exists in App Store Connect
- [x] **TF-03**: Public TestFlight link is active
- [x] **TF-04**: README on GitHub includes public TestFlight link with install instructions
