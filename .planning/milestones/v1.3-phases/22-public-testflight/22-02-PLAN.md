```markdown
# Dictus v1.3.0-beta -- Public Beta

Dictus is a free, open-source iOS keyboard with on-device voice dictation. This is the first public beta.

## What's new in v1.3

### Rebuilt keyboard (zero dead zones)
The entire keyboard has been rebuilt from scratch using UICollectionView (previously SwiftUI). Every key responds on first tap, including edge keys like A, Q, P, and M.

### Improved touch interactions
- Haptic feedback on every key press (matches Apple keyboard feel)
- Key popup preview on press
- Hold backspace for accelerating delete
- Long-press vowels for French accent characters (drag to select)
- Drag spacebar to move cursor (trackpad mode with haptic ticks)
- Adaptive accent key: shows apostrophe after consonants, accent after vowels
- 3-category key sounds (letter, delete, modifier) respecting silent switch

### Autocorrect and suggestions
- 3-slot suggestion bar with French autocorrect
- Tap to insert, backspace to undo autocorrect
- Autocorrect-on-space matches iOS native behavior

### Bug fixes
- Dynamic Island no longer gets stuck on REC state
- Export logs completes in seconds with progress indicator

## Install

**[Join the TestFlight beta](https://testflight.apple.com/join/XXXXX)**

Requirements: iPhone 12+, iOS 17.0+

## Feedback

- Shake your device in Dictus to send TestFlight feedback
- Or open a [GitHub issue](https://github.com/getdictus/dictus-ios/issues)

## Privacy

All speech recognition runs on-device. No data leaves your phone. [Privacy Policy](https://www.getdictus.com/en/privacy)
```
